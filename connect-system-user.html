Conectar Página com Token Permanente

Use um System User Token do Meta Business para conectar de forma permanente, sem expiração.

1Acesse o Meta Business Suite
Vá em business.facebook.com/settings/system-users e selecione o Business Portfolio que tem suas páginas.
2Crie um System User
Clique em Add → Create system user:
  • Name: Wozza Integration
  • Role: Admin
3Atribua páginas ao System User
Com o System User selecionado, clique em Add Assets → Pages e selecione a(s) página(s) que quer conectar com permissão Full Control.
4Gere o token permanente
Ainda no System User, clique em Generate New Token:
  • App: wozza-social-monitor
  • Token expiration: Never
  • Permissions (marque todas):
  • instagram_basic
  • instagram_manage_comments
  • instagram_manage_insights
  • pages_show_list
  • pages_read_engagement
  • pages_read_user_content
  • business_management
Clique Generate Token e copie o token gerado (aparece só uma vez!).
5Cole o token aqui
Validando...
6Selecione a página que deseja conectar
